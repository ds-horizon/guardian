## Key Features

- **Passwordless Authentication**: Secure authentication using OTP via SMS, Email, or other channels
- **Password-based Auth**: Traditional username/password authentication with strong security
- **Social Login**: OAuth-based authentication with Google, Facebook, and other providers
- **Multi-tenant**: Built-in multi-tenancy support for complex applications
- **Token Management**: JWT-based access and refresh tokens with rotation support
- **Security First**: Built with security best practices and compliance in mind

## Why Guardian?

Guardian provides a comprehensive authentication and authorization solution that combines security, flexibility, and ease of use. Whether you're building a mobile app, web application, or API service, Guardian has you covered.

### Multiple Authentication Methods

- **Passwordless**: Eliminate password vulnerabilities with OTP-based authentication
- **Password-based**: Traditional authentication for applications that require it
- **Social Authentication**: Seamless integration with popular identity providers
- **Guest Access**: Allow temporary access without full registration

### Enterprise Features

- **Multi-tenancy**: Isolated environments for different organizations
- **Role-based Access Control**: Fine-grained permission management
- **Audit Logging**: Complete audit trail of authentication events
- **Webhook Support**: Real-time notifications for critical events

### Developer Friendly

- **Simple SDK**: Easy-to-use client libraries for multiple platforms
- **Comprehensive Documentation**: Detailed guides and API references
- **Active Development**: Regular updates and new features
- **Open Source**: Community-driven development

## Quick Start

Get up and running in minutes:

```typescript
import { Guardian } from '@guardian/sdk';

const guardian = new Guardian({
  tenantId: 'your-tenant-id',
  apiUrl: 'https://guardian-api.dreamsportslabs.com'
});

// Passwordless authentication
const { state } = await guardian.passwordless.init({
  flow: 'signinup',
  contacts: [{
    channel: 'sms',
    identifier: '+1234567890'
  }]
});

const { accessToken } = await guardian.passwordless.complete({
  state,
  otp: '123456'
});
```
